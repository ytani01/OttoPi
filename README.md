# OttoPi

OTTO like robot for Raspberry Pi Zero

## Install

### start pigpiod

$ sudo pigpiod


### install pipenv

$ sudo pip3 install -U pip
$ hash -r

$ sudo pip3 install pipenv

$ pipenv install


## run command in virtualenv

$ pipenv run <command> <arg> ..


## uninstall

### remove virtualenv

$ pipenv --rm
